## Controls
###### Player 1 - Samurai Mack

  **a** - *move left*
  **d** - *move right*
  **w** - *jump*
  **space** - *attack*

###### Player 2 - Kenji

  **arrow left** - *move left*
  **arrow right** - *move right*
  **arrow up** - *jump*
  **p** - *attack*
